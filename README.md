# Install / run info

Unity project:

- open project in unity, then go into "Assets/Scenes" and manually open "SampleScene" to be able to run the project.

App (APK)

- follow the direct link to google cloud [AR_FurnitureApp (Android only)](https://drive.google.com/file/d/1lS2regMiAfWpaTbye07y424JJuLS7tB7/view)
- download (most likely prompted if u trust the app)
- after download, to install the app u will get a popup again telling u to go to settings and allow downloads from unsafe sources (if u haven't already)
- after enabling downloads from unsafe sources, u can install the app.
- when starting the app, it will ask for camera and recording permissions, the game runs without this aswell, but u won't be able to see ur own area inside the game unless u allow camera permissions.
